اینجا من قراره 3 تاکانکشن بیارم بالا و اون سه تا کانکشن رو بریج کنم توی 3 تا شبکه، بعد از اون میفرستم برا میکروتیک روتر



اول سافت اتر رو نصب میکنیم

```

./Install.sh

```
